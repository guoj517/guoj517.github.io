Consider ways that you might organize the 100 items. Come up with at least 5 distinct ideas for a set of 3 sub-categories.

  • Visual Similarities (styles)

  • Number of People

  • Colored vs B&W

  • Series

  • Theme (e.g. nature)

Consider how this method of organizing might be conceptually connected to the content / image collection

  • Visual Similarities (styles): Pleasing to look at, looks great if organized in grids. Can be used to explore the pinterest idea.

  • Number of People: Shows the difference between individual behaviors and interaction. Make the collection more insightful.

  • Colored vs B&W: Also great for visual presentaton.

  • Series & Theme: editorial element. Exihibition.
